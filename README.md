# 拼音小课堂

这是一个适合小学生使用的汉语拼音练习 H5 程序，已经按手机微信浏览器做了界面适配。

练习内容包括：

- 从音节中找声母
- 从音节中找韵母
- 判断一声、二声、三声、四声
- 看汉字选择正确拼音

## 本机预览

在当前目录运行：

```bash
python3 -m http.server 4173
```

电脑浏览器打开：

```text
http://127.0.0.1:4173
```

## 手机微信局域网测试

手机和电脑连接同一个 Wi-Fi。

查看电脑局域网 IP：

```bash
ipconfig getifaddr en0
```

假设显示 `192.168.1.23`，手机微信里打开：

```text
http://192.168.1.23:4173
```

如果打不开，通常是手机和电脑不在同一个 Wi-Fi，或者电脑防火墙拦截了局域网访问。

## 微信里直接打开的正式方式

微信好友或群聊里直接点开，需要把这些文件部署到一个公网 HTTPS 地址：

- `index.html`
- `styles.css`
- `script.js`
- `icon.svg`
- `manifest.webmanifest`

可选部署方式：

- GitHub Pages
- Netlify
- Vercel
- 腾讯云 COS 静态网站托管
- 阿里云 OSS 静态网站托管

部署完成后，把 HTTPS 链接发到微信，手机即可直接打开使用。

## 如果要做成真正的微信小程序

还需要：

- 注册微信小程序账号
- 安装微信开发者工具
- 把当前 H5 逻辑改造成小程序的 `wxml`、`wxss`、`js` 结构
- 提交微信审核后发布

当前版本是 H5 版，优势是开发快、能通过链接分享；小程序版更正式，但发布流程更长。
